Smarty plugin

Load JavaScript.

Examples:

  1.  Load jquery library 1.2.1
  2.  <wa:js src="/lib/jquery/1_2_1.js" />
  3.  translate to:
  4.  <script type="text/javascript" src="/pub/lib/jquery/1_2_1.js"></script>
  5.  
  6.  Load custom javascript which located /pub/js/admin/edit_user.js
  7.  <wa:js src="/js/admin/edit_user.js" />
  8.  translate to:
  9.  <script type="text/javascript" src="/pub/js/admin/edit_user.js"></script>
  10.  
  11.  Load external javascript
  12.  <wa:js src="http://example.com/example.js" />
  13.  translate to:
  14.  <script type="text/javascript" src="http://example.com/example.js"></script>
  15.  
  16.  Load IE only javascript
  17.  <wa:js src="/js/admin/fix_layout.js" ie="yes" />
  18.  translate to:
  19.  <!--[if IE]>
  20.  <script type="text/javascript" src="/pub/js/admin/fix_layout.js"></script>
  21.  <![endif]-->
  22.  
  23.  Load javascript on IE which version less than 7
  24.  <wa:js src="/lib/ie7/ie7-standard-p.js" ie="yes" version="lt 7" />
  25.  <!--[if lt IE 7]>
  26.  <script type="text/javascript" src="/pub/lib/ie7/ie7-standard-p.js"></script>
  27.  <![endif]-->
