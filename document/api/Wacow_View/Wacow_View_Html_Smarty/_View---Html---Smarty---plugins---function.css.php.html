Smarty plugin

Load CSS.

Examples:

  1.  Load custom style sheet which located /pub/css/admin/user.css
  2.  <wa:css href="/css/admin/user.css" />
  3.  convert to:
  4.  <link rel="stylesheet" type="text/css" href="/pub/css/admin/user.css" media="all" />
  5.  
  6.  Load style sheet of jquery thickbox for screen
  7.  <wa:css href="/lib/jquery/thickbox.css" media="screen" />
  8.  convert to:
  9.  <link rel="stylesheet" type="text/css" href="/pub/lib/jquery/thickbox.css" media="screen" />
  10.  
  11.  Load external javascript
  12.  <wa:css href="http://example.com/example.css" />
  13.  convert to:
  14.  <link rel="stylesheet" type="text/css" href="http://example.com/example.css" media="all" />
  15.  
  16.  Load IE only style sheet
  17.  <wa:css href="http://example.com/example.css" ie="yes" />
  18.  convert to:
  19.  <!--[if IE]>
  20.  <link rel="stylesheet" type="text/css" href="http://example.com/example.css" media="all" />
  21.  <![endif]-->
  22.  
  23.  Load javascript on IE which version less than 7
  24.  <wa:css href="/theme/wacow/style_ie.css" ie="yes" version="lt 7" />
  25.  <!--[if lt IE 7]>
  26.  <link rel="stylesheet" type="text/css" href="/pub/theme/wacow/style_ie.css" media="all" />
  27.  <![endif]-->
